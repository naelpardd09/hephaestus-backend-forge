# Requirements

Berikut kebutuhan tools dan teknologi untuk mengikuti pembelajaran di repository ini.

## Bahasa dan Framework

- Java 8
- Spring 5.3.24

## Database dan Koneksi

- Oracle Database 19c
- Oracle JDBC OJDBC8
- HikariCP 5.0.1

## Cache dan Infrastruktur

- Redis 5.0.3
- Docker 28.0.4
- Docker Compose

## Development Tools

- IntelliJ IDEA atau VS Code
- Postman
- Git
